選択した課題: 演習1_b 配列の並び順を逆順にする関数を作成する

方針: 
一度、配列aの内容を配列saveに保存をし、
お互いに逆の方向から参照をしていく。
その参照をした際に配列saveの要素を配列aに上書きすることで、
逆順になるようにする

コード: 
void iarray_revese(int *a, int n) {
  int save[n];
  for(int i = 0; i < n; ++i) {
    save[i] = a[i];
  }
  for(int i = 0; i < n; ++i) {
    a[i] = save[n-i-1];
  }
}

実行例:
OK 85241 -> 14258
  1  4  2  5  8
  1  4  2  5  8

解説:
方針通りに作成した
その際に、for文を二度活用することで作成に成功した

レビュー課題:
forを二度やると実行時間も長くなるため、保存するのは配列aの前半部分で十分なのではないか

残念ながら、今回の課題で新しく発見したことはなかった
